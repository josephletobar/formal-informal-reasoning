# Cover Letter Draft

**To:** The Editors, *Nature Machine Intelligence*  
**Re:** Manuscript submission, “Reusable Internal Computation in Language Models: An ABC Test with Attribution Graphs and Causal Interventions”

Dear Editors,

We submit the enclosed Article for consideration in *Nature Machine Intelligence*.

The manuscript asks a focused mechanistic question: when a language model solves the same simple computation in explicit, applied and implicit forms, do those tasks recruit a reusable internal computation? We combine a controlled ABC benchmark with behavioral controls, representation comparisons, attribution graphs, sparse-feature interventions, multilayer path tests, attention-source interventions and residual-state transfer in larger models.

The main contribution is methodological and scientific. The Gemma 2 2B attribution graphs show strong recurrence within matched prompt forms, and graph-selected paths and source positions influence answer-directed logit margins. However, a discovery/confirmation split with active same-position controls does not show held-out necessity for the recurrent sparse features, and cross-form activation transfer does not establish operation-specific sufficiency. Larger-model residual screens show compatibility between same-computation states, while making clear that residual compatibility is not sparse circuit localization. By reporting these positive and negative results together, the study provides a falsification-oriented framework for separating recurring answer-directed structure from evidence for a reusable reasoning module.

We believe this distinction is relevant to the journal's interest in machine intelligence, interpretability and the mechanisms underlying generalization. The work does not claim that discrete reasoning modules have been established. Instead, it provides a reproducible benchmark, explicit causal controls and a set of results that identify which stronger claims are and are not supported by current evidence.

The manuscript is not published elsewhere and is not under consideration by another journal. All authors have approved the submission. The public research release contains the benchmark generator, prompt manifest, derived raw analysis tables, figure-generation code, environment record and reproducibility checks; the serialized attribution graphs are retained in the authors' local archive and can be supplied during review. Model weights and gated credentials are not redistributed.

Thank you for considering this work.
